# MyBlog

مدونة شخصية بسيطة باستخدام **GitHub Pages**.

## 📄 الصفحات
- **index.html** : الصفحة الرئيسية
- **about.html** : صفحة "عني"
- **blog.html** : صفحة المدونة
- **contact.html** : صفحة التواصل

## 🎨 التصميم
- تم استخدام ملف **style.css** لتنسيق الصفحات.
- يمكنك تعديل الألوان والخطوط بسهولة من خلال هذا الملف.

## 🚀 النشر
- الموقع منشور تلقائيًا عبر GitHub Pages على الرابط:
  [https://jalalthawaba.github.io/myblog](https://jalalthawaba.github.io/myblog)

## ✨ المميزات
- تصميم بسيط وسهل التعديل
- إمكانية إضافة صفحات جديدة بسهولة (مثلاً services.html)
- يعمل مباشرة على الإنترنت بدون استضافة خارجية

## 🛠️ كيفية التطوير
إذا أردت تعديل أو تطوير المدونة:
1. انسخ المستودع (Fork) أو نزّله إلى جهازك.
2. عدّل ملفات HTML أو CSS حسب رغبتك.
3. ارفع التغييرات (Commit + Push) إلى الفرع `main`.
4. سيتم نشر التغييرات تلقائيًا عبر GitHub Pages.

يمكنك إضافة صفحات جديدة (مثل services.html) أو تعديل التصميم في ملف style.css.
